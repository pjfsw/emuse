JOFMODORE SE SBC & emulator
========================================================
Files related to JOFMODORE SE (Sixty Eight), my 68000-based single board computer.

Hardware specifications
-----------------------
* 68HC000FN clocked at 12 MHz
* 1 MB RAM 
* 64 KB (EEP)ROM
* MMC/SD-card reader
* SNES controller connector
* External SPI-port
* External peripheral-port (8-bit)

Planned additions
----------------
* VGA graphics card/audio via peripheral port
* FAT-16 filesystem and autoboot from MMC

Emulator
--------
Per instruction emulator of the 68000 CPU with autovector interrupt emulation
